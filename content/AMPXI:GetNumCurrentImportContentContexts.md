# Request

| Index Word | Description                |
|------------|----------------------------|
| 0          | Header code \[0x00250000\] |

# Response

| Index Word | Description                               |
|------------|-------------------------------------------|
| 0          | Header code                               |
| 1          | Result code                               |
| 2          | Number of Current Import Content Contexts |

# Description

Returns the number of import content contexts that are active for the current title installation.
